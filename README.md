## 使用说明
1. 在Release或者[官网](https://www.eebbk.tbit.top)下载最新版本的工具箱7z压缩文件/Setup.exe
2. 解压缩文件/安装程序
3. 运行（压缩文件请运行run.bat，安装程序直接打开即可）
4. 按照提示操作
